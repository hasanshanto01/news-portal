Most Common Frequently Asked Question Answer

Var can be updated and re-declared into the global scope. It can be declared without initialization. Let can be updated but cannot be re-declared into the block scope. It can be declared without initialization. Const cannot be updated or re-declared into the block scope. It cannot be declared without initialization.

Regular functions created using function declarations or expressions are constructible and callable. Since regular functions are constructible, they can be called using the new keyword. However, Arrow functions are only callable and not constructible, i.e arrow functions can never be used as constructor functions.

You can use template strings to write your string exactly how you want it to appear, without all the quotation marks and confusion. Template strings make it very easy to spot a missing space and make your string more readable.

.map() executes the same code on every element in an array and returns a new array with the updated elements. .forEach() is used to execute the same code on every element in an array but does not change the array and it returns undefined. .filter() checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria. .find() checks every element in an array to see if it meets a certain criteria and returns the first element that return truthy for the criteria.
